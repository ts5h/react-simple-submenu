# React Simple Submenu
A simple submenu with CSS3 for React-based web pages.  

## Live Demo
https://ts5h.github.io/react-simple-submenu/

## Getting Started

### Installation
```sh
cd react-simple-submenu
yarn install
yarn start
```

### Other Options
See ```package.json```
